    分屏编辑文件

    打开多个文件

    命终端模式下

    vim filename1 filename2

    命令行模式下

    :open filename

    打开文件间切换

    快捷键

    # 下一个文件
Ctrl + 6

    命令行

    # 下一个文件
:bn
# 上一个文件
:bp

    分窗格显示多个文件

    # 纵向分屏显示
:split
# 横向分屏显示
:vsplit

    多窗格间切换

    快捷键

    # 顺序切换下一个窗格
Ctrl + ww
# 掐换到前/下/上/后一个窗格
Ctrl + h/j/k/l
